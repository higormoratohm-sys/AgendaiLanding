# AgendAí

## Sobre o projeto

O AgendAí tem como objetivo facilitar a conexão entre profissionais de estética e seus clientes, oferecendo uma plataforma simples e prática para agendamentos de serviços. O aplicativo permite que barbeiros, manicures, cabeleireiros e outros profissionais divulguem seus serviços, gerenciem horários e atendam com mais organização, enquanto os usuários encontram e reservam atendimentos de forma rápida e conveniente.

O site tem como propósito apresentar o projeto ao público, explicar suas funcionalidades e benefícios, e direcionar tanto clientes quanto profissionais para download ou cadastro. Ele funciona como uma vitrine oficial do aplicativo, destacando sua proposta, recursos principais e facilitando o primeiro contato com a plataforma.

## Como executar?

### Pré-requisitos

Antes de começar, você vai precisar instalar:

- [Bun](https://bun.com/)(v1.2 ou superior)

### Testando em modo de desenvolvimento

Abra seu terminal (PowerShell, CMD ou Bash) e execute os comandos:

```
$ bun install # instala dependências
```

Agora, você pode testar o projeto com o comando:

```
$ bun dev
```

Um link para acessar o site localmente será disponibilizado em seu terminal.
